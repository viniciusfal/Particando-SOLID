

### 🎲 Rodando a aplicação

```bash
# Instale as dependências
$ yarn

# Execute a aplicação
$ yarn dev
```
<strong> Para ver a documentação consulte o seu <a href='#'>localhost:3333/api-docs</a> </strong>
